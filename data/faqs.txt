What are the prerequisites for machine learning?
To learn machine learning you need to know Python programming and basic mathematics including linear algebra and statistics.
How long does it take to learn data science?
Learning data science typically takes around 6 to 12 months with consistent practice and dedication.
What is the difference between machine learning and deep learning?
Machine learning uses traditional algorithms to learn from data while deep learning uses neural networks with many layers.
What is Python used for in data science?
Python is used for data analysis data visualization building machine learning models and deploying applications.
What is a neural network?
A neural network is a computational model inspired by the human brain that learns patterns from data through training.
What is supervised learning?
Supervised learning is a type of machine learning where the model learns from labeled training data to make predictions.
What is unsupervised learning?
Unsupervised learning is a type of machine learning where the model finds patterns in data without labeled examples.
What is overfitting in machine learning?
Overfitting occurs when a model learns the training data too well and fails to generalize to new unseen data.
How do you prevent overfitting?
You can prevent overfitting by using regularization techniques dropout cross validation and collecting more training data.
What is a decision tree?
A decision tree is a machine learning algorithm that makes decisions by splitting data based on feature values.
What is the difference between classification and regression?
Classification predicts discrete categories while regression predicts continuous numerical values.
What is a confusion matrix?
A confusion matrix is a table used to evaluate the performance of a classification model showing true and false predictions.
What is cross validation?
Cross validation is a technique to evaluate machine learning models by training and testing on different subsets of the data.
What is gradient descent?
Gradient descent is an optimization algorithm that minimizes the loss function by iteratively updating model parameters.
What is the learning rate in machine learning?
The learning rate is a hyperparameter that controls how much the model weights are updated during each training step.
What is feature engineering?
Feature engineering is the process of creating new features from raw data to improve the performance of machine learning models.
What is natural language processing?
Natural language processing is a branch of artificial intelligence that enables computers to understand and generate human language.
What is transfer learning?
Transfer learning is a technique where a model trained on one task is reused as the starting point for a different related task.
What is the LSTM network?
LSTM stands for Long Short Term Memory and it is a type of recurrent neural network designed to learn from sequential data.
How does text generation work?
Text generation works by training a model on a corpus of text and then predicting the most likely next word given an input sequence.
What is tokenization?
Tokenization is the process of converting text into individual tokens such as words or characters for processing by a model.
What is word embedding?
Word embedding is a representation of words as dense numerical vectors that capture semantic relationships between words.
What is the softmax function?
The softmax function converts a vector of raw scores into a probability distribution over multiple classes.
What is categorical cross entropy?
Categorical cross entropy is a loss function used for multi class classification problems that measures prediction error.
What is an epoch in machine learning?
An epoch is one complete pass through the entire training dataset during the model training process.
What is batch size?
Batch size is the number of training samples processed before the model weights are updated during training.
What is the Adam optimizer?
Adam is an adaptive learning rate optimization algorithm that combines the benefits of gradient descent with momentum.
How do you save a machine learning model?
You can save a machine learning model using frameworks like TensorFlow Keras by calling the save method on the model object.
What is model inference?
Model inference is the process of using a trained machine learning model to make predictions on new input data.
What is a vocabulary in NLP?
A vocabulary in NLP is the set of unique words or tokens that the model knows and can process from the training data.
What is sequence padding?
Sequence padding is the process of adding zeros or special tokens to sequences to make them all the same length for batch processing.
How does autocomplete work?
Autocomplete works by using a trained language model to predict the most likely next word given the words already typed by the user.
What is beam search?
Beam search is a decoding algorithm that explores multiple possible continuations simultaneously to find the best sequence.
What is greedy decoding?
Greedy decoding selects the most probable token at each step without considering future tokens in the sequence.
What is temperature in text generation?
Temperature is a parameter that controls the randomness of predictions with lower values making output more deterministic.
What is top k sampling?
Top k sampling selects the next token randomly from the top k most probable tokens to add diversity to generated text.
